### 👿 과제 제출 가이드
```
1. 매주 [주차/이름]의 브랜치를 생성합니다. ex. week1/soomin

2. 본인 이름의 폴더 내부의 프로젝트 파일 안에서 해당 주차 실습을 진행합니다.

3. main 브랜치와 contribute 하여 PR을 작성합니다.

4. PR의 내용은 자세할수록 좋습니다.

5. main 브랜치를 pull 받아 최신화를 유지해주세요.
```
